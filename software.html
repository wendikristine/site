Software

What is Available?

This list represents our most popular software.

VASP

This first-principles quantum chemistry/physics molecular simulation package calculates electronic structure and dynamics.

R

With a strong online support community, this statistical programming language also supports software development.

Geant4

Primarily used for high-energy physics, this software simulates the movement of particles through matter using Monte Carlo methods.

Fortran

The scientific programming standard, this language has been actively developed for more than half a century!

Gaussian

Boasting a long list of methods, functionals, and basis sets, this quantum simulation package boasts a user interface.

Python

Specifically designed for scientific computing, this programming language is verstile and great for beginners due to simplified syntax.

Bioconductor

Mostly available as R modules, this tool package focuses on analyzing genomic data with enhanced visualization options.

ROOT

Developed by scientists at CERN, ROOT is a data analysis framework that can be integrated with R and Python.
